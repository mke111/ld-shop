# 加密货币入门指南

本指南介绍主流加密货币的基础知识和实操流程。

## 一、比特币/以太坊原理简介

### 比特币（Bitcoin/BTC）

比特币是第一个去中心化加密货币，2009年由中本聪创建。

**核心原理**
- **区块链技术**：所有交易记录保存在分布式账本中，无法篡改
- **工作量证明（PoW）**：通过矿机算力竞争来确认交易
- **总量恒定**：总量2100万枚，永不增发
- **去中心化**：无央行发行，由全球节点共同维护

**主要特点**
- 匿名性：地址与真实身份无直接关联
- 不可逆：交易一旦确认无法撤销
- 可分割：最小单位聪（0.00000001 BTC）
- 全球流通：无国界限制

### 以太坊（Ethereum/ETH）

以太坊是第二代区块链，支持智能合约。

**核心原理**
- **智能合约**：自动执行的程序代码
- **以太坊虚拟机（EVM）**：运行智能合约的环境
- **权益证明（PoS）**：持币即可参与验证，无需矿机
- **Gas费用**：执行操作需要支付ETH作为燃料

**主要特点**
- DeFi（去中心化金融）基础设施
- NFT（非同质化代币）标准
- DApp（去中心化应用）生态
- 可编程性强

### 两者对比

| 特性 | 比特币 | 以太坊 |
|------|--------|--------|
| 定位 | 数字黄金 | 世界计算机 |
| 共识 | PoW（将转向PoS） | PoS |
| 可编程 | 有限 | 完全可编程 |
| 速度 | 慢（7 TPS） | 较快（15-30 TPS） |
| 应用 | 储值 | 金融应用、NFT |

## 二、Binance注册与KYC流程

Binance（币安）是全球最大的加密货币交易所。

### 注册流程

1. **访问官网**
   - 网址：binance.com（注意识别钓鱼网站）
   - 建议从搜索引擎或朋友推荐获取链接

2. **创建账户**
   - 点击「注册」
   - 建议使用邮箱注册（不要用手机号）
   - 设置强密码（至少8位，包含大小写和数字）

3. **安全设置**
   - 开启两步验证（Google Authenticator或短信）
   - 建议绑定邮箱和手机
   - 设置防钓鱼码

### KYC实名认证

1. **进入认证页面**
   - 登录后点击「用户中心」→「身份认证」

2. **选择认证级别**
   - **基础认证**：仅需邮箱和手机号，提取限额低
   - **完整认证**：需要身份证，限额高

3. **完整认证步骤**
   1. 选择国家/地区（中国大陆）
   2. 选择证件类型（身份证/护照）
   3. 上传身份证正反面照片
   4. 进行人脸识别验证
   5. 等待审核（通常几分钟到几小时）

4. **注意事项**
   - 确保证件照片清晰
   - 光线充足，避免反光
   - 按照提示做相应动作

### 交易前准备

1. **充值方式**
   - C2C交易：与个人用户买卖
   - 快捷买币：银行卡直接购买（需要海外银行卡）
   - 充值USDT：從其他交易所转入

2. **基本操作**
   - 现货交易：买入/卖出加密货币
   - 合约交易：杠杆交易（风险高）
   - 法币交易：C2C交易

## 三、MetaMask钱包安装配置

MetaMask是以太坊生态最流行的钱包。

### 安装步骤

1. **浏览器插件**
   - Chrome/Edge/Brave/Firefox浏览器
   - 访问 metamask.io
   - 点击「Download」→选择浏览器
   - 添加到Chrome

2. **手机App**
   - App Store或Google Play下载
   - 搜索「MetaMask」

3. **创建钱包**
   - 点击「Create a new wallet」
   - 设置密码（至少8位）
   - 观看助记词视频（了解重要性）
   - **非常重要**：记录12个英文单词的助记词

4. **备份助记词**
   - 手写在纸上（不要截图或截图加密存储）
   - 多备份几份
   - 存放在安全位置

### 网络配置

MetaMask默认连接以太坊主网，添加其他网络：

**Arbitrum One**
- 网络名称：Arbitrum One
- RPC URL：https://arb1.arbitrum.io/rpc
- ChainID：42161
- 符号：ETH
- 浏览器：https://arbiscan.io

**Optimism**
- 网络名称：Optimism
- RPC URL：https://mainnet.optimism.io
- ChainID：10
- 符号：ETH
- 浏览器：https://optimistic.etherscan.io

**Polygon**
- 网络名称：Polygon
- RPC URL：https://polygon-rpc.com
- ChainID：137
- 符号：MATIC
- 浏览器：https://polygonscan.com

### 基本使用

1. **收发代币**
   - 点击账户地址复制
   - 点击「发送」输入对方地址和数量

2. **添加代币**
   - 点击「导入代币」
   - 输入代币合约地址

3. **连接DApp**
   - 点击「连接钱包」
   - 授权即可访问去中心化应用

## 四、助记词安全存储方法

助记词是钱包的「root密码」，丢失等于丢失全部资产。

### 最佳实践

1. **物理备份**
   - 手写在纸上（不要用打印机）
   - 使用铅笔+纸更耐久
   - 多抄写几份

2. **分散存储**
   - 不要全部放在一个地方
   - 家里、公司、银行保险箱各放一份

3. **金属备份**
   - 购买金属助记词板（Cryptosteel等）
   - 防火防水防腐

### 绝对不要做

- ❌ 截图保存
- ❌ 存放在手机/电脑
- ❌ 发送给任何人（包括客服）
- ❌ 存放在云盘/邮箱
- ❌ 在联网设备上明文存储

### 助记词验证

创建钱包后，尝试删除钱包重新导入，确保备份正确。

## 五、现货交易基础

现货交易是指即时买卖加密货币。

### 订单类型

**限价单（Limit Order）**
- 设置买入/卖出价格
- 只有达到价格才成交
- 优点：可控制成本
- 示例：当前价格$3000，设置为$2900买入

**市价单（Market Order）**
- 以当前最优价格立即成交
- 优点：快速成交
- 缺点：可能滑点

**止损单（Stop-Loss）**
- 设置触发价格
- 当价格达到时自动卖出
- 用于控制亏损

### 交易术语

- **Taker**：吃单方，立即成交
- **Maker**：挂单方，等待成交
- **Maker手续费通常更低**
- **滑点**：实际成交价与预期价格的差异
- **深度**：订单簿的厚度，代表流动性

### 新手建议

1. **只投资能承受损失的资金**
2. **分散投资，不要All in**
3. **学习基本面和技术分析**
4. **从小额开始**
5. **设置止盈止损**

## 六、常见骗局识别

### 常见骗局类型

**1. 钓鱼网站**
- 伪造交易所/钱包官网
- 域名相似（如binance-com.io）
- 诱导输入账户密码

**防范**
- 手动输入网址
- 检查域名
- 开启钓鱼防护

**2. 假交易所**
- 诱导充值后无法提现
- 初期可正常交易
- 大额提现时冻结

**防范**
- 选择主流交易所
- 查证交易所背景
- 小额测试提现

**3. 假客服**
- 通过Telegram/Discord主动联系
- 声称能帮忙解决问题
- 要求提供助记词

**防范**
- 官方不会主动联系
- 永远不透露助记词

**4. 杀猪盘**
- 恋爱/友情为幌子
- 诱导投资虚假平台
- 先给甜头，最后一刀

**防范**
- 不要相信陌生人的投资建议
- 独立判断

**5. 假空投**
- 声称免费送代币
- 需要连接钱包授权
- 授权后盗走资产

**防范**
- 不随意授权陌生合约
- 检查代币官网

**6. 庞氏骗局**
- 承诺高额固定收益
- 拉人头获得奖励
- 本质是资金盘

**防范**
- 远离「稳赚不赔」
- 收益率超过10%要警惕

### 安全建议

1. 开启交易所的所有安全验证
2. 资产分散存放（热钱包+冷钱包）
3. 不相信任何主动联系的「客服」
4. 仔细核对每一步操作
5. 学习基础区块链知识

## 总结

加密货币入门要点：
- 比特币是数字黄金，以太坊是智能合约平台
- 交易所选Binance等主流平台
- 钱包助记词是核心，务必安全备份
- 现货交易从限价单开始
- 警惕一切高收益承诺

投资有风险，入市需谨慎。
