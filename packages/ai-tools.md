# AI工具入门指南

本指南将帮助你快速掌握主流AI工具的注册和使用方法。

## 一、ChatGPT注册步骤

ChatGPT是OpenAI推出的AI对话工具，注册需要海外手机号验证。

### 注册流程

1. **准备工作**
   - 准备一个邮箱（Gmail/Outlook均可）
   - 准备一个海外手机号（见下文推荐）
   - 稳定上网环境（VPN/代理）

2. **获取虚拟手机号**
   - **TextNow**：下载App，注册后获取美国虚拟号码
   - **Google Voice**：在Google账户中申请美国号码
   - **sms-activate.org**：付费接码平台，支持ChatGPT验证

3. **注册步骤**
   1. 访问 chat.openai.com
   2. 点击Sign Up，输入邮箱
   3. 选择邮箱验证或Google/Apple账户登录
   4. 输入收到的验证码
   5. 输入虚拟手机号，获取验证码完成验证
   6. 填写基本信息即可开始使用

### 注意事项
- 免费版GPT-3.5足够日常使用
- GPT-4需要Plus订阅（$20/月）
- 国内用户需确保网络稳定

## 二、Claude.ai注册使用

Claude是Anthropic推出的AI助手，以安全性和长文本处理著称。

### 注册流程

1. 访问 claude.ai
2. 点击Get Started，选择Google登录或邮箱注册
3. 同样需要海外手机号验证（可用TextNow）
4. 验证通过后即可使用

### 版本选择
- **Claude 3 Haiku**：免费，速度快
- **Claude 3 Sonnet**：平衡性能与速度
- **Claude 3 Opus**：最强能力，适合复杂任务

## 三、常用Prompt模板20条

### 写作类（5条）

1. **文章润色**
   ```
   请帮我润色以下文章，使其更加流畅专业：[文章内容]
   ```

2. **标题生成**
   ```
   为以下内容生成5个吸引人的标题：[主题]
   ```

3. **大纲创建**
   ```
   为[主题]创建一篇详细的文章大纲
   ```

4. **内容扩展**
   ```

   ```

5. **文案撰写**
   ```
   帮我写一段针对[目标人群]的[产品/服务]推广文案
   ```

### 编程类（5条）

6. **代码解释**
   ```

   ```

   ```
   以下代码报错[错误信息]，请帮我修复：[代码]
   ```

8. **代码优化**

10. **技术方案**

## 四、Midjourney入门指南

Midjourney是AI图像生成工具，通过Discord机器人使用。

### 使用步骤

1. **注册Discord**
   - 访问 discord.com 注册账号
   - 下载客户端或使用网页版

2. **加入Midjourney**
   - 访问 midjourney.com
   - 点击Join Beta，加入Midjourney服务器

3. **基本命令**
   - `/imagine`：生成图片
   - `/info`：查看账户信息
   - `/settings`：调整设置
   - `/help`：获取帮助

4. **提示词结构**
   ```
   /imagine prompt: [描述词] --ar 16:9 --v 6 --s 750
   ```
   - `--ar`：宽高比
   - `--v`：版本号
   - `--s`：风格化程度

5. **操作流程**
   - 在频道输入框输入 `/imagine prompt:`
   - 描述你想要生成的图像
   - 等待生成（大约1分钟）
   - 使用U1-U4放大图片，V1-V4生成变体
